How to Make a Fully Functional Level


About

This game is constructed column by column.
So by analogy, the level input is also formatted column by column.
The game does not do any sanity checks on the level input.
So if there is an alphabetic character somewhere,
the game crashes when the offending character is read.

To make level design easier, a Python program is provided.
Do not use "degenerator.py". Use "generator.py".
